# GUIDE_outputs.md

## Purpose

This folder stores artifacts generated by the correlation-analysis stage.

Inputs still come from root `outputs/`:

- `data/raw/volume_screen.csv`
- `data/raw/daily_close_volume_screened_2016_2025.parquet` for notebook workflows

This folder now owns:

- method selections such as `selected_ward.csv`
- the merged consensus file `selected_merged.csv`
- diagnostics such as `heatmap_*.png`, `dendrogram_*.png`,
  `greedy_marginal_diversity.png`, and `maxdiv_marginal_dr.png`
